# 📊 Руководство импорта/экспорта данных

## 🎯 Быстрый старт

### Экспорт товаров
```bash
# Войти в админ-панель: http://localhost/admin
# 1. Перейти на http://localhost/admin/api/product/
# 2. Нажать кнопку "Export" (вверху справа)
# 3. Выбрать формат (CSV или XLSX)
# 4. Скачать файл
```

### Импорт товаров
```bash
# 1. Перейти на http://localhost/admin/api/product/
# 2. Нажать кнопку "Import" (вверху справа)
# 3. Загрузить CSV или XLSX файл
# 4. Проверить превью данных
# 5. Нажать "Confirm Import"
```

## 📋 Структура CSV файлов

### Товары (products.csv)

```csv
id,name,description,self_price,price,image_directory
1,Кожа черная,Качественная черная кожа для кошельков,1000.00,1100.00,leather/black
2,Кожа коричневая,Премиум коричневая кожа,1500.00,1650.00,leather/brown
3,Кожа красная,Натуральная кожа ярко-красного цвета,1200.00,1320.00,leather/red
```

**Поля:**
- `id` - ID товара (опционально при создании, автоматически при обновлении)
- `name` - Название товара (обязательно)
- `description` - Описание товара (опционально)
- `self_price` - Себестоимость товара (обязательно)
- `price` - Цена с наценкой (рассчитывается автоматически, можно оставить пустым)
- `image_directory` - Папка с фото в `/media/` (опционально)

### Заказы (orders.csv)

```csv
id,full_name,phone,mail,address,order_date,summary_price,is_approve,is_paid
1,Иван Петров,+7-999-123-45-67,ivan@mail.com,"ул. Ленина, 10, кв. 5",2025-05-06 10:30,5500.00,1,1
2,Мария Сидорова,+7-999-987-65-43,maria@mail.com,"пр. Пушкина, 20",2025-05-05 14:15,3300.00,1,0
```

**Поля:**
- `id` - ID заказа (не редактируется)
- `full_name` - ФИО клиента
- `phone` - Телефон клиента
- `mail` - Email клиента
- `address` - Адрес доставки
- `order_date` - Дата заказа (формат: YYYY-MM-DD HH:MM)
- `summary_price` - Сумма заказа
- `is_approve` - Проверен (0 или 1)
- `is_paid` - Оплачено (0 или 1)

### Заявки на заказ (custom_orders.csv)

```csv
id,full_name,phone,mail,order_date,comment,is_approve,is_paid
1,Алексей Иванов,+7-999-555-55-55,alex@mail.com,2025-05-06 09:00,"Нужна кожа под заказ: размер 50x50см",1,0

```

## 🔄 Действия при конфликтах

При импорте можно выбрать стратегию для существующих записей:

| Действие | Описание |
|----------|---------|
| **Skip** | Пропустить существующие записи |
| **Create** | Создать новые (если такой ID есть, ошибка) |
| **Update** | Обновить существующие записи |

## 📝 Примеры использования

### Пример 1: Добавить новые товары

**Создайте файл `new_products.csv`:**
```csv
name,description,self_price,image_directory
Кожа оранжевая,Ярко-оранжевая натуральная кожа,950.00,leather/orange
```

2. Загрузите файл

**Экспортируйте товары, отредактируйте:**

**Импортируйте с "Update":**
- Старые цены заменятся на новые

## 🐛 Troubleshooting

**Проблема:** "KeyError: 'name'"
- **Решение:** Убедитесь что в файле есть колонка "name"

**Проблема:** "Цена не обновляется"
- **Решение:** Цена рассчитывается автоматически на основе self_price + 10%

**Проблема:** "Фотографии не привязываются"
- **Решение:** Убедитесь что папка существует в `/back/media/`

**Проблема:** "Ошибка кодировки при импорте"
- **Решение:** Сохраните CSV в кодировке UTF-8

## 📚 Полезные форматы

### UTF-8 кодировка в Excel
1. Открыть → Сохранить как
2. Тип файла: CSV (разделители-запятые) (*.csv)
3. Кодировка: UTF-8

### Экспорт из Google Sheets
1. File → Download → Comma-separated values (.csv)
2. Google автоматически использует UTF-8

### Экспорт из LibreOffice
1. File → Save As
2. Format: Text CSV (.csv)
3. Character set: UTF-8
